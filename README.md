# StreamDPS Final Overlay

Bu paket, kullanicinin onayladigi son tasarima gore hazirlandi:
- seffaf overlay
- ustte geri sayim
- 4 takim karti
- hedef yok
- sure bitince en yuksek skor kazaniyor
- kazanan animasyonu
- kontrol paneli ayri sayfada
- lider takim degistikce takim sarkisi kaldigi yerden devam ediyor

## Takim / hediye eslesmesi
- FB -> Gul / Rose
- GS -> TikTok
- BJK -> GG
- TS -> Dondurma Kulahi / Ice Cream Cone

Her hediye varsayilan olarak +1 puan ekler.

## Kurulum
```bash
npm install
npm start
```

## Sayfalar
- Overlay: `http://localhost:3000/`
- Kontrol paneli: `http://localhost:3000/control.html`

## Ses dosyalari
`public/audio/` klasorune su dosyalari koy:
- `fb.mp3`
- `gs.mp3`
- `bjk.mp3`
- `ts.mp3`

Not: Cogu tarayici otomatik ses oynatmayi kisitlar. Bu nedenle overlay ilk acildiginda bir kez tiklamak gerekebilir.
